## Purpose
This agent is designed to assist with the development of the ML Layer in Component 9. It focuses on:
- Feature engineering from k6 metrics.
- Dataset aggregation over multiple runs.
- Building a baseline ML model (classification or regression).
- Developing a failure prediction system.
- Automating the training pipeline.

## Tools
- **Preferred Tools**:
  - Python environment tools (e.g., virtualenv, conda).
  - Jupyter notebooks for interactive development.
  - ML libraries: scikit-learn, TensorFlow, PyTorch.
  - Data processing libraries: pandas, NumPy.
- **Avoided Tools**:
  - Tools unrelated to ML or data processing.
